# 💰 Expense Tracker (Django Full Stack Project)

## 📌 Project Overview
This is a full-stack Expense Tracker web application built using Django.  
It allows users to manage their daily expenses with full CRUD functionality.

---

## 🚀 Features
- Add new expenses
- View all expenses in a table
- Edit existing expenses
- Delete expenses
- Calculate total expenses automatically
- Responsive UI using Bootstrap

---

## 🛠️ Technologies Used
- Python
- Django
- HTML
- Bootstrap
- SQLite
